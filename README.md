# 噗浪潑你冷水機器人

前身是DC機器人 ([傳送門](https://github.com/kinako890419/discord_chatbot))

## 功能

| qualifier | 關鍵字         | 備註          |
| --------- | ----------- | ----------- |
| 希望        | `!抽`        | 抽三張塔羅牌      |
| 想要        | `!抱怨`       |             |
| 問         | `!為什麼`      |             |
| 問         | `!要不要`      | 隨機決定選項+解釋理由 |
| -         | `要不要`、`好不好` | 只有回覆隨機的決定選項 |
| -         | 其他關鍵字       |             |

驚嘆號全形或是半形皆可
